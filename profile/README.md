# About

政府「資安即國安」之戰略目的，因應新型態資安威脅，國立成功大學鏈結產官學研資源，提升跨域實務教學能力，攜手數所大專校院、法人單位、國營企業以及民間企業，建置油、水、電等三項關鍵基礎設施教學基地與資安實證場域，共同培育民生能源之資安人才，提升國家關鍵基礎設施資安實戰量能。
